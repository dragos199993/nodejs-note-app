# Node JS Note app CLI

# How to use?
First run:
```sh
$ npm install
```
To add:
```sh
$ node index.js add --title="My first note" --body="This is the content of the first note" 
```
To remove:
```sh
$ node index.js remove "My first note"
```
To get item:
```sh
$ node index.js get "My first note"
```

To list all notes:
```sh
$ node index.js list
```
